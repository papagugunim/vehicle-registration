# 🚛 차량 입차 등록 시스템
Lotte KF 러시아법인 차량 입차 관리 시스템

## 📋 기능

### 1. 차량 정보 등록
- 날짜, 거래처명, 주문번호
- 화물차량 정보 (헤드/트레일러 번호)
- 운전자 정보 (이름, 여권, 전화번호)
- 입차 예정 시각
- 입차 완료 여부

### 2. 등록 현황 조회
- 전체/오늘/대기/완료 필터
- 실시간 검색 (차량번호, 거래처, 주문번호, 운전자명)
- 입차 상태 토글
- 본인 등록 항목만 수정/삭제 가능

### 3. 사용자 관리
- 영업 매니저 (10명)
- 물류팀 (10명)
- 창고 직원 (5명)
- 경비 (3명)

### 4. 권한 관리
- 입력자 정보 자동 기록
- 본인이 등록한 차량만 수정/삭제
- 설정 페이지 비밀번호 보호 (1234)

## 🚀 설치 방법

### 1. Google Sheets 설정
1. Google Sheets에서 새 스프레드시트 생성
2. `확장 프로그램` > `Apps Script` 클릭
3. `google-apps-script.js` 파일 내용 복사하여 붙여넣기
4. `배포` > `새 배포` 클릭
5. 유형: `웹 앱` 선택
6. 실행 권한: `나` 선택
7. 액세스 권한: `누구나` 선택
8. 배포 URL 복사

### 2. 시스템 설정
1. `index.html` 파일을 웹 서버에 업로드
2. 시스템 접속 후 `설정` 탭 클릭
3. 비밀번호 `1234` 입력
4. Google Sheets URL에 배포 URL 붙여넣기
5. 사용자 목록 설정
6. 저장

## 📱 사용 방법

### 차량 등록
1. `차량 등록` 탭 클릭
2. 입력자 이름 선택
3. 차량 정보 입력
4. `등록하기` 버튼 클릭

### 등록 현황 확인
1. `등록 현황` 탭 클릭
2. 필터 선택 (전체/오늘/대기/완료)
3. 검색창에 키워드 입력하여 검색

### 입차 완료 처리
1. `등록 현황`에서 해당 차량 찾기
2. `완료로 변경` 버튼 클릭

### 수정/삭제
1. 본인이 등록한 차량만 가능
2. `수정` 버튼: 등록 폼으로 이동하여 수정
3. `삭제` 버튼: 확인 후 삭제

## 🔒 보안

- 설정 페이지는 비밀번호(1234)로 보호
- 본인이 등록한 차량만 수정/삭제 가능
- 모든 등록 이력 자동 기록

- **클라우드**: Google Sheets에 자동 동기화
